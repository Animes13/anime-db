# Anime DB (TMDB)

Banco de dados público de animes baseado no TMDB.

## Fonte
- The Movie Database (TMDB)

## Conteúdo
- Anime TV
- Anime Movies
- Atualização automática diária

## Estrutura
- data/anime_tv.json
- data/anime_movies.json
- data/meta.json

## Uso
Ideal para addons Kodi, scrapers e catálogos públicos.

## Atualização
Automática via GitHub Actions.
